## English

### Overview

Cfly is an open-source desktop workflow automation toolkit for streamlining development and deployment pipelines.

It provides reusable workflow components and configurations to help you automate common tasks on your desktop, integrate with CI/CD systems, and keep your development process efficient and consistent.

### 简介

Cfly 是一个桌面端开源工作流自动化工具，用于简化开发和部署流程。
